# Java Practice

This repository contains my Java practice code, organized into packages.

## Structure

The source code is located in `src/com/nitish/practice` and organized into the following packages:

- **arrays**: Array manipulation and algorithms.
- **math**: Mathematical operations and number theory.
- **basics**: Basic Java concepts (loops, operators).
- **utils**: Utility classes.

## How to Run

To compile the project:
```bash
javac -d out -sourcepath src src/com/nitish/practice/*/*.java
```

To run a specific file (e.g., Deletion):
```bash
java -cp out com.nitish.practice.arrays.Deletion
```
